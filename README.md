<p align="center" >
  Very simple Outfit-Store but there's some creativity;
  <br>
  Which I designed it to be fast and have no item-details page with an on-the-fly_add-to-cart button.

<p align="center" >

  <br>
  • Checkout validation
  <br>
  • Delete item
  <br>
  • Responsive
